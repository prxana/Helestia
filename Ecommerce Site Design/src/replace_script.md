# Script de remplacement pour la couleur beige crème

## Fichiers à modifier
Les fichiers suivants doivent avoir toutes les occurrences de "white" remplacées par "#fffaf2":

1. /pages/Cart.tsx
2. /pages/Checkout.tsx
3. /pages/Contact.tsx
4. /pages/Customization.tsx
5. /pages/OrderConfirmation.tsx
6. /pages/ProductDetail.tsx
7. /pages/Profile.tsx

## Remplacements à effectuer
- `text-white` → `text-[#fffaf2]`
- `bg-white` → `bg-[#fffaf2]`
- `border-white` → `border-[#fffaf2]`
- `hover:text-white` → `hover:text-[#fffaf2]`
- `hover:bg-white` → `hover:bg-[#fffaf2]`
- `hover:border-white` → `hover:border-[#fffaf2]`
- `focus:border-white` → `focus:border-[#fffaf2]`
- `placeholder:text-white` → `placeholder:text-[#fffaf2]`
